Nodejs 安装 ​

  1. 去官网下载对应包

    https://nodejs.cn/download/

  2. cmd 测试 是否安装对应版本

    shell
    node -v
npm -v

  3. 配置npm 文件夹

    1.找到安装的目录，在安装目录下新建两个文件夹【node_global】和【node_cache】

    2.使用管理员身份打开cmd 配置对应这俩个文件夹

    shell
    npm config set cache "D:\nodejs\node_cache"

npm config set prefix "D:\nodejs\node_global"

-- 查看当前地址
npm config get prefix

npm config get cache

  4. 配置环境变量

    1. 在【系统变量】中点击【新建】 安装路径

    变量名：NODE_PATH

    变量值：C:\Program Files\nodejs\node_global\node_modules

    1. 编辑【用户变量】中的【Path】

    将默认的 C 盘下【 AppData\Roaming\npm 】修改成 D:\nodejs\node_global 【node_global】的路径

    1. 在【系统变量】中选择【Path】点击【编辑】添加 %NODE_PATH%

  5. 设置源（这里选择淘宝源 也可以选择别的）

    shell
    -- 最新的淘宝镜像  （https://registry.npm.taobao.org） 已经停止维护
npm config set registry https://registry.npmmirror.com

-- 查看当前镜像 
npm config get registry
